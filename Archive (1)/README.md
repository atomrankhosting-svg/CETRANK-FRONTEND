# CETRANK — AI-Powered Admission Intelligence Engine

Navigate Maharashtra CET counselling with AI-powered predictions, real-time cutoff analysis, and personalized college recommendations.

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion + GSAP
- Three.js
